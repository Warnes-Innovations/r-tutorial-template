# 01_documents/

Background materials, data governance documents, and reference files for this project.

## What belongs here

| Type | Examples |
|------|---------|
| Data dictionaries / codebooks | `data_dictionary_nndss.md`, `codebook_ehr_extract.xlsx` |
| Data use agreements | `DUA_signed_2024.pdf` |
| Protocol / methods memos | `analysis_protocol_v1.docx` |
| Background reports | `cdc_std_surveillance_report_2023.pdf` |
| Meeting notes | `2024-03-15_stakeholder_notes.md` |
| Variable recoding decisions | `recode_decisions_race_ethnicity.md` |

## What does NOT belong here

- Raw data files → `02_raw_data/`
- Analysis scripts → `scripts/`
- Output reports → `06_output_docs/`

## Naming convention

Use `YYYY-MM-DD` prefixes for dated documents and descriptive lowercase names with underscores:

```
2024-03-15_stakeholder_notes.md
data_dictionary_nndss_2024.md
dua_state_cdc_signed.pdf
```
